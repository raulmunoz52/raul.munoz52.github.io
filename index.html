Grado Medio: Sistemas Microinformáticos y Redes

Resumen de los módulos

Montaje y Mantenimiento de Equipos

Aprenderás a montar, reparar y mantener ordenadores y periféricos, aplicando normas de seguridad y calidad.

Sistemas Operativos Monopuesto

Instalación, configuración y mantenimiento de sistemas operativos como Windows y Linux en entornos personales.

Aplicaciones Ofimáticas

Manejo avanzado de programas de oficina: procesadores de texto, hojas de cálculo, presentaciones y bases de datos.

Redes Locales

Diseño, instalación y configuración de redes de área local (LAN), incluyendo cableado, switches y configuración IP.

Sistemas Operativos en Red

Instalación y administración de sistemas operativos en red como Windows Server o Linux, creando usuarios y recursos compartidos.

Seguridad Informática

Protección de sistemas frente a amenazas, gestión de copias de seguridad y configuración de antivirus y firewalls.

Servicios en Red

Configuración de servicios de red como DNS, DHCP, servidores web y FTP en entornos Linux y Windows.

Aplicaciones Web

Diseño y creación básica de páginas web usando HTML, CSS y fundamentos de programación web.

Empresa e Iniciativa Emprendedora

Conocimientos básicos sobre creación de empresas, trabajo autónomo, financiación y gestión de proyectos.

Formación y Orientación Laboral (FOL)

Derechos laborales, prevención de riesgos laborales y orientación para la búsqueda activa de empleo.

Formación en Centros de Trabajo (FCT)

Prácticas en empresas para aplicar los conocimientos adquiridos en un entorno real.
